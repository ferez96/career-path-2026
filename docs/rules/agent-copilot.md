
# Copilot — dev & project framework

- Read `docs/AGENT_ROLES.md` (Copilot section) before changing structure or policy.
- Anything intended for public: classify files (`docs/DATA_CLASSIFICATION.md`), merge gate per `docs/BRANCH_WORKFLOW.md`.
- Large binaries: follow `.gitattributes` (LFS); do not use LFS for `.md/.csv/.json/.yaml` unless there is a documented reason.
- Commits/PRs: note decision mode and sanitize scope when publish-related.
