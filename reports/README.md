# Reports (`derived-sanitized`)

Chỉ chứa **output đã khử PII** dùng cho public hoặc context agent.

| Thư mục | Nội dung |
|---------|----------|
| `benchmarks/` | Phân tích benchmark / JD (fit, gap, khuyến nghị) |
| `briefs/` | Tóm tắt công ty / bối cảnh (không-PII) |
| `private/` | Báo cáo chỉ dùng local — gitignored |

Raw input luôn nằm trong `data/raw/` (gitignored), không commit vào đây.
